## FE-1: 暗色/亮色双主题系统

- [ ] 定义 CSS 变量体系：暗色 `:root` + 亮色 `:root.light`，覆盖所有颜色 token（背景、面板、卡片、输入框、边框、文字、强调色、成功/失败/警告）
- [ ] 实现 `useTheme()` composable，管理主题状态（默认暗色），持久化到 `localStorage`
- [ ] Header 右上角添加主题切换按钮（月亮/太阳图标），点击切换 `html.classList`
- [ ] 所有组件使用 CSS 变量而非硬编码颜色，确保主题切换无闪烁
- [ ] Element Plus 组件跟随主题：暗色用 `dark` 模式，亮色用默认模式

**原型参考：** `:root` 第 10-31 行，`:root.light` 第 32-51 行，`toggleTheme()` 函数

---

## FE-2: 处理器卡片网格选择器

- [ ] 处理器列表渲染为 2 列网格卡片，每张卡片含：emoji 图标 + 处理器名称 + 简短描述
- [ ] 点击选中时高亮（紫色边框 + 发光阴影），其余卡片取消选中
- [ ] 切换导航标签时，按 `data-category` 过滤只显示当前分类的处理器
- [ ] 默认选中当前分类的第一个处理器
- [ ] 处理器图标在后端 `params_schema` 中新增 `icon` 字段定义，前端读取渲染

**原型参考：** `.processor-grid` + `.processor-card`，`switchTab()` 过滤逻辑

---

## FE-3: 分类联动参数面板

- [ ] 根据当前选中的导航标签（图片/视频/音频），切换显示对应的参数表单面板
- [ ] 每个分类有独立的参数面板 DOM（`params-image`、`params-video`、`params-audio`）
- [ ] 切换标签时隐藏非当前分类的参数面板，显示当前分类的
- [ ] 参数面板内的表单控件仍由 `ParamForm.vue` 根据 `params_schema` 动态渲染
- [ ] 分类切换同时更新上传区域的文件格式提示文字（如图片显示"PNG、JPG、WebP"，视频显示"MP4、AVI、MOV"）

**原型参考：** `params-image`、`params-video`、`params-audio` 三个面板，`uploadHints` 对象

---

## FE-4: 上传文件预览卡片

- [ ] 文件上传成功后，上传区域下方显示文件信息卡片
- [ ] 卡片内容：缩略图（图片生成缩略图，视频/音频显示类型图标）+ 文件名 + 文件大小 + 分辨率/时长
- [ ] 卡片右侧显示删除按钮（×），点击可移除已上传文件
- [ ] 拖拽上传时上传区域高亮（虚线边框变色 + 背景色变化）

**原型参考：** `.file-card`、`.upload-zone.dragover` 样式

---

## FE-5: 处理中遮罩层（Overlay）

- [ ] 点击"开始处理"后，右侧预览区覆盖半透明遮罩层
- [ ] 遮罩层内容：双圈旋转 Spinner + 百分比数字（大号）+ 当前步骤文字
- [ ] 百分比和文字随 WebSocket 进度实时更新
- [ ] 任务完成/失败后遮罩自动消失（0.6s 过渡动画）

**原型参考：** `.processing-overlay`、`.spinner`、`startProcess()` 动画逻辑

---

## FE-6: 原图/结果对比预览

- [ ] 预览区分为左右两栏，左侧"原图"、右侧"结果"
- [ ] 每栏顶部有标签（"原图"灰色，"结果"绿色），使用毛玻璃背景
- [ ] 原图在上传后立即显示，结果在处理完成后显示
- [ ] 无上传文件时显示空状态占位（图标 + 提示文字 + 网格背景）
- [ ] 预览区背景使用径向渐变 + 网格线装饰

**原型参考：** `.preview-compare`、`.preview-side`、`.preview-empty`、`.preview-area::before` 网格

---

## FE-7: 流光进度条

- [ ] 进度条使用渐变色（紫 → 粉 → 紫），背景大小 200%，CSS 动画实现流光滚动效果
- [ ] 宽度随 WebSocket 进度百分比实时变化（CSS transition 平滑过渡）
- [ ] 任务完成时进度条变为绿色，停止动画

**原型参考：** `.progress-bar-fill`、`@keyframes shimmer`

---

## FE-8: 控制台日志面板

- [ ] 底部固定高度（160px）的可滚动日志区域，等宽字体显示
- [ ] 每条日志格式：时间戳 + 消息内容
- [ ] 消息按类型着色：info（紫色）、success（绿色）、error（红色）、普通（灰色）
- [ ] 面板顶部标题栏显示"控制台" + 当前状态标签（处理中/已完成/失败）
- [ ] 状态标签颜色与消息类型一致
- [ ] 新日志自动滚动到底部

**原型参考：** `.log-panel`、`.log-body`、`.log-msg.info/.success/.error`

---

## FE-9: 背景粒子动画

- [ ] 全页面固定背景层，生成 20 个随机大小（2-6px）的圆形粒子
- [ ] 粒子从底部向上浮动，随机速度和延迟，颜色随机（紫色/粉色）
- [ ] 粒子透明度通过 CSS 变量控制（暗色 15%，亮色 8%）
- [ ] `pointer-events: none` 不影响交互
- [ ] 尊重 `prefers-reduced-motion` 系统设置，动效减弱时禁用

**原型参考：** `.particles`、`.particle`、`@keyframes float`

---

## FE-10: Header 导航栏渐变装饰线

- [ ] Header 底部添加 1px 渐变装饰线（透明 → 紫色 → 粉色 → 透明）
- [ ] 导航标签栏使用圆角容器包裹，激活态使用紫蓝渐变背景 + 发光阴影
- [ ] Logo 使用渐变文字效果（`background-clip: text`）

**原型参考：** `.header::after`、`.nav-tab.active`、`.logo`

## 实施建议

2. **FE-2**（处理器卡片）和 **FE-3**（参数联动）随 Task 23 通用组件一起做
3. **FE-4**（文件卡片）和 **FE-6**（对比预览）随 Task 23-24 做
4. **FE-5**（遮罩层）、**FE-7**（流光进度条）、**FE-8**（控制台）随 Task 23 做
5. **FE-10**（Header 装饰）随 Task 21 布局框架做
